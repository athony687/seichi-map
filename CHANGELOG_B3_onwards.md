# B3以降の変更まとめ

## 地図スタイル・ピン調整

- **B3**: 地図スタイル・テーマカラー統一、カスタムピン（紫星型）導入
- 聖地ピンのサイズ縮小（scale 1.6 → 1.0）
- 地図カラーをポップ配色 → 黄色ベース → クリームトーンに段階調整

## 観光スポット機能

- 主要観光スポット10か所（通天閣・清水寺・東大寺・金閣寺・伏見稲荷・嵐山・道頓堀・奈良公園・姫路城・大阪城）をオレンジ丸ピンで追加
- クリックで「日本語名（英語名）」のポップアップ表示
- データを `tourist_spots.json` に分離（コードから独立して編集可能）
- ピンサイズ縮小（scale 8 → 5）
- 白画面クラッシュ修正（`google.maps.SymbolPath.CIRCLE` → 数値 `0` に変更）

## デモ機能強化

- スクラバー（スライダー）追加：デモルートを任意の位置に手動でシーク可能
- 近接距離を 120m → 10km → 3km に調整
- カード表示ロジック改善：3km圏内で維持、離れたら自動クローズ（チカチカ防止）
- スクラバー操作時のチカチカ修正（triggeredRef をクリアしないよう変更）
- 近接スポットのパルスアニメーション追加（ピンが拡縮して目立つ）

## カードUI改善

- カードをデフォルト折りたたみ表示に変更
- タップ/クリックで本文を展開する「Read more」ボタン追加
- 「Read more」ボタンを半透明ピル形状で強調表示
- スマホでも地図が見えるよう、展開前はコンパクトに収まる設計

## PWA対応（ホーム画面インストール）

- `manifest.json` 追加（アプリ名・テーマカラー・アイコン設定）
- `index.html` にPWAメタタグ追加（apple-touch-icon・theme-color）
- アイコン画像（192x192 / 512x512）生成・調整
  - 元画像 `map_trial.png` からコンテンツ領域をタイトクロップ
  - コーナーの灰色背景をテーマカラー紫（#7c3aed）に置換
  - iOS ホーム画面で余白感が出ないよう最終調整
